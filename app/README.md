# Application

## Backend

Node.js + Express.js

Generated by:
```bash
npm init
npm install --save express body-parser
npm install --save-dev nodemon
```

Run:
```bash
npm start
```

## Frontend

React

Generated by:
```bash
npx create-react-app frontend
```

### Additional dependencies

#### React Router
```bash
npm install --save react-router-dom@5 --save-exact
```

Run:
```bash
npm start
```
